
# PockeQube

#### Installation

##### **Requirements**

* XCode 12
* Cocoapods installed 

##### **Install**

```
pod install --repo-update 
```

##### **Update**

```
pod update
```

#### Execute 

Open PocketQube project (Pockecube.xcworkspace) and select run (menu or ctrl + R)
